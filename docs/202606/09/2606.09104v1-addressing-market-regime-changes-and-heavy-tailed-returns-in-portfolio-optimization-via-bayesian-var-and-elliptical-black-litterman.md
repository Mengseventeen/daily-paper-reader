## 摘要
用于投资组合优化的深度强化学习框架因其能够从市场数据动态学习分配规则而显示出前景。然而，这些模型未能考虑实际市场行为中极端事件更频繁的特征——肥尾收益。此外，历史数据被同质化处理，未考虑时间重要性，导致模型在市场机制变化时失效。我们提出了一种新的BAVAR-BLED算法，它在TD3架构内结合了源自贝叶斯平均向量自回归和基于椭圆分布的Black-Litterman模型的方法。BAVAR捕获一组考虑多尺度时间特征的向量自回归表示，使得基于对市场机制敏感的预期收益和离散度矩阵的估计进行自适应分配决策成为可能。这些估计作为BLED的先验输入，BLED模型使用学生t分布，从而获得更现实的肥尾收益估计。BAVAR-BLED算法使用Transformer网络进行观点构建，使用CNN进行风险厌恶估计，根据市场条件修改动态分配决策。对29个道琼斯工业平均指数成分股在长达十年的市场期间的评估表明，BAVAR-BLED显著优于最先进的方法，实现了1.72的夏普比率和2.70的索提诺比率，总收益率为57.26%。

## Abstract
Deep reinforcement learning (DRL) frameworks for portfolio optimization have shown promise for their ability to learn allocation rules dynamically from market data. However, these models fail to account for fat-tailed returns, which characterize actual market behavior with more frequent extreme events. Furthermore, historical data is treated homogeneously, without accounting for temporal importance, leading models to fail during regime changes. We propose a new BAVAR-BLED algorithm that combines methods derived from Bayesian-Averaging Vector Autoregressive (BAVAR) and the Black-Litterman model using Elliptical Distributions (BLED) within a TD3 architecture. BAVAR captures a set of vector autoregressive representations that consider multi-scale temporal features, enabling adaptive allocation decisions based on regime-aware estimates of return expectations and dispersion matrices. These estimates serve as prior inputs to BLED, a model that uses Student's t-distributions, allowing for more realistic fat tail return estimates. The BAVAR-BLED algorithm uses transformer networks for view construction and CNNs for risk-aversion estimates, which modify dynamic allocation decisions based on market conditions. An evaluation of 29 Dow Jones Industrial Average constituents over a decade-long market period shows that BAVAR-BLED significantly outperforms state-of-the-art methods, achieving Sharpe and Sortino ratios of 1.72 and 2.70, respectively, and total returns of 57.26%.

## 1. 核心问题与整体含义（研究动机和背景）

现有的深度强化学习（DRL）投资组合优化框架主要存在两个根本缺陷：
- **忽视收益的肥尾（重尾）分布特征**：实际金融市场中极端事件发生频率远高于正态分布假设，导致传统方法低估尾部风险，在市场动荡时可能产生巨大损失。
- **同质化处理历史数据**：大多数方法（包括作者之前的BLED模型）在计算市场先验（期望收益和离散矩阵）时，将历史观测视为同等重要，无法适应市场制度（regime）的突然变化（如COVID-19危机、利率周期等）。

本文提出的 **BAVAR-BLED** 框架，通过将贝叶斯平均向量自回归（BAVAR）与椭圆Black-Litterman模型（BLED）集成到TD3深度强化学习架构中，首次同时解决了上述两个问题，实现了对市场制度变化的自适应和肥尾收益的更真实建模。

## 2. 方法论：核心思想与关键技术细节

**核心思想**：利用BAVAR维护一个VAR模型集合，通过贝叶斯学习动态更新每个模型的后验，并基于预测似然加权平均得到制度感知的先验期望收益 \(\mu\) 和离散矩阵 \(D\)。该先验替代了传统BLED中基于全部历史数据静态计算的先验，再输入到椭圆Black-Litterman后验计算中与Transformer生成的观点及CNN估计的风险厌恶系数结合，最终得到最优投资组合权重。

**关键技术细节（算法流程）**：

1. **状态表示**：每步 \(t\) 的状态 \(s_t\) 是维度 \(29\times15\times12\) 的张量（29只资产，15日回溯窗口，12个特征：价格、成交量、EMA、MACD、RSI、布林带等）。

2. **三路并行分支**：
   - **BAVAR模块**：提取HAR特征（日、周、月收益率，即1、5、22个交易日），维护 \(\approx 600\) 个VAR模型。每个模型通过贝叶斯递归更新系数和精度，并基于预测似然更新模型权重。最终加权平均输出 \(\mu\) 和 \(D\)。
   - **Transformer编码器**：根据状态张量生成投资者观点向量 \(Q\)（n维期望收益）。
   - **CNN风险网络**：估计状态依赖的风险厌恶系数 \(\delta\)。

3. **BLED计算**：采用学生t分布（椭圆分布族）建模收益。后验期望收益和离散矩阵按Black-Litterman公式融合先验与观点：
   \[
   \mu_{BL} = \mu + [(\tau D)^{-1} + P^\top \Omega^{-1} P]^{-1} P^\top \Omega^{-1} (Q - P\mu)
   \]
   \[
   D_{BL} = D - D P^\top (Q + P D P^\top)^{-1} P D
   \]
   最优权重 \(w^* = \frac{1}{\delta} D_{BL}^{-1} \mu_{BL}\)。

4. **TD3训练**：演员网络对BLED权重进行微调；双Q网络减少过估计偏差；延迟策略更新；经验回放缓冲。BAVAR更新与DRL训练分离（异步每月更新12次，减少计算开销）。
