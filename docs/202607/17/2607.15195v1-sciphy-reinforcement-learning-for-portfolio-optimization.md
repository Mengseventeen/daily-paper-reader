## 摘要
本文提出了一种面向大型机构投资者的动态投资组合优化框架，采用科学物理信息强化学习（SciPhyRL）方法。该框架在包含显式累积成本的扩展状态空间上以连续时间形式构建，利用离线历史数据学习最优的、具有分布感知能力的策略。一项核心创新在于：通过将优化问题投影到观测轨迹上，将其转化为路径型Hamilton-Jacobi方程，从而简化为求解HJB方程。该方程通过PINN直接从数据中一步离线求解，无需传统的值迭代或策略迭代。为使该方法在实际短时域中有效，控制变量从连续交易速率重新定义为离散目标持仓量。这确保了信号隐含的头寸能够立即达到，同时执行成本基于微观结构基础的二次价格影响模型进行评估。在一个包含14只ETF的资产池上，使用工程化的预言机信号进行评估，学习得到的吉布斯策略在样本外夏普比率上相比静态和短视基线有显著提升。结果表明，所提出的框架成功将已知信号质量转化为稳健的、多周期的、成本感知的配置机制，并严格控制波动率和换手率。

## Abstract
This paper introduces a dynamic portfolio optimization framework for large institutional investors using Scientific Physics-Informed Reinforcement Learning (SciPhyRL). Formulated in continuous time over an extended state space that includes explicit cumulative costs, the approach leverages offline historical data to learn optimal, distribution-aware strategies. A core innovation reduces the optimization challenge to solving an HJB equation by projecting it onto observed trajectories as a pathwise Hamilton-Jacobi equation. This is solved directly from data using PINN in a single offline sweep, eliminating the need for traditional value or policy iteration. To make the method effective at practical short horizons, the control variable is recast from a continuous trading rate to a discrete target holding. This ensures signal-implied positions are reached immediately, while execution costs are evaluated against a microstructure-grounded quadratic price impact model. Evaluated on a $14$-asset ETF universe using an engineered oracle signal, the learned Gibbs policy yields substantial out-of-sample Sharpe ratio improvements over static and myopic baselines. The results demonstrate that the proposed framework successfully translates known signal quality into a robust, multi-period, and cost-aware allocation mechanism with strictly controlled volatility and turnover.

### 2. 论文提出的方法论

- **核心思想**：
    1.  **分布性离线连续时间强化学习 (DOCTR-L)**：采用分布性的、熵正则化的强化学习视角，将优化问题从传统的最小化期望总成本，扩展到管理总回报的整个条件分布。
    2.  **路径Hamilton-Jacobi (HJ) 方程**：将原始的HJB方程（一个高阶非线性偏微分方程）投影到可观测的历史轨迹上，简化为一个一阶路径依赖的HJ方程。这类似于特征线法，每个样本路径充当特征曲线。
    3.  **物理信息神经网络 (PINN) 求解**：用神经网络参数化值函数 `Jθ`，并设计一个损失函数来强制满足路径HJ方程。同时，利用精确的Girsanov似然比将行为策略的数据重新加权为最优策略的示范，将**离线RL问题转化为一个单一的监督学习任务**，无需传统的值迭代或策略迭代。
    4.  **控制变量重新参数化**：为了在短时间周期内有效，将连续控制变量（交易速率 `at`）重新定义为离散的目标持仓量 `ht`。这使得信号隐含的头寸可以立即达到，提高了信号捕获效率。

- **关键技术细节与公式**：
    - **扩展状态空间**：状态变量 `y_t = (x_t, S_t, C_t)`，分别代表持仓、资产价格和累积成本。
    - **二次型成本与动态**：由于市场冲击的存在，运行成本和资产价格漂移均与控制变量 `at` 呈二次型关系，问题超出了经典的LQR框架。
    - **HJB方程推导**：从熵正则化的成本泛函出发，推导出最优值函数 `J` 满足的HJB方程。
    - **Hamilton-Incepted (HI) 方程**：通过对HJB方程进行解析最小化，得到一个简化的半线性偏微分方程，称为HI方程，其中非线性部分被吸收到配分函数 `Z` 中。
    - **高斯混合模型 (GMM)**：假设行为策略 `π0` 是一个高斯混合模型，这使得最优策略 `π*` 的配分函数可以解析求解，保持吉布斯策略的GMM结构。
    - **路径HJ方程**：沿最优轨迹应用伊藤引理，将二阶项抵消，得到一阶的路径HJ方程，免去了对二阶导数的直接求解。
    - **损失函数**：最终的PINN损失函数由两部分构成：
        - **模型损失**：强制路径HJ残差为零。
        - **似然比项**：约束由模型隐含的价格动态与观测数据一致，通过最优策略的Girsanov似然比将行为数据重加权。
- **算法流程**：
    1.  收集历史轨迹数据 `{ (x_t, S_t, C_t) }`。
    2.  构建一个神经网络 `Jθ` 来近似值函数。
    3.  定义一个损失函数，该函数在每条轨迹上强制实现路径HJ方程，并包含利用Girsanov似然比将行为策略转化为最优策略的项。
    4.  通过优化损失函数来训练神经网络 `Jθ`。
    5.  从训练好的 `Jθ` 中解析地计算出最优的吉布斯策略 `π*`。
