## 摘要
由于金融市场中可用投资选项的不断增加，决策对投资者构成了日益严峻的挑战。过去几十年的一个热门研究领域是投资组合优化，其旨在确定投资者应在哪些资产上投资多少。在优化模型中引入现实条件会使问题变为NP难问题，精确方法求解效率低下；因此，研究人员转向进化算法来近似求解。本文提出了多目标进化算法的增强策略，该策略能够在基数约束下的投资组合优化问题中提供更快的收敛速度和广泛的搜索能力。为了实现这些特性，引入了一种独特的解表示方法、一种新型算子以及新的修复机制来解决上述问题，其中对投资组合中的资产数量设置了上下限。为此，结合上述策略的新交配方案被应用于知名的多目标进化算法中，以求解该问题。随后，使用著名的市场指数作为基准，将定制算法与传统算法进行了比较。结果表明，所提出的策略不仅能提供更好的近似解，而且收敛速度更快，并且随着市场中资产数量的增加，性能不会下降。

## Abstract
Decision-making is posing an increasingly formidable challenge to investors because of the growing number of alternatives available in financial markets. A hot area of research over the past few decades has been portfolio optimization that seeks to determine how much an investor should invest in which asset. Introducing real-world conditions to the optimization model turns the problem into an NP-hard one for whose solution exact methods become inefficient; hence, researchers have turned to evolutionary algorithms to approximate solutions. In this paper, strengthening strategies are presented for multi-objective evolutionary algorithms that can provide a faster convergence rate and extensive search ability in the portfolio optimization problem under the cardinality constraint. To implement those features, a unique solution representation, a novel operator, and new repair mechanisms are introduced for solving the aforementioned problem in which lower and upper limits are set on the number of assets in the portfolio. For this purpose, new mating strategies along with the aforesaid package are implemented in well-known multi-objective evolutionary algorithms to solve the problem. The customized algorithms are subsequently tested against traditional ones using well-known market indices as benchmarks. Results indicate that the proposed strategy not only provides better approximations but also converges faster as well at no loss of performance with an increasing number of assets in the market.

## 1. 核心问题与整体含义（研究动机和背景）

- **研究问题**：解决基数约束下的投资组合优化问题（Cardinality-Constrained Portfolio Optimization Problem, CCPOP），即在给定资产池中选取有限数量的资产，并分配资金，以同时最大化收益和最小化风险。
- **背景与动机**：现代投资组合理论（Markowitz均值-方差模型）忽略了许多现实约束（如基数约束：规定资产数量上下限）。引入基数约束后，问题变为NP-hard，精确方法（如分支定界）效率低下。因此，研究者转向进化算法（如多目标进化算法）寻求近似解。然而，现有方法在处理大规模资产（如数百支股票）时收敛慢、搜索能力不足。本文旨在提出增强策略，加速收敛并提高解质量，同时保持性能随资产数量增长不下降。

### 关键技术细节

1. **解表示**：染色体由两行组成，长度固定为Kmax。第一行存储选中的资产索引（允许重复），第二行对应每个位置的资金比例。重复资产的权重由其各位置的权重之和表示。这种编码自动保证了资产数不超过Kmax，仅需处理低于Kmin的情况。

2. **交配策略**：
   - **膝点锦标赛选择（前期α%迭代）**：基于到乌托邦点（收益最大、风险最小）的欧氏距离，选择靠近膝点的解配对，产生优秀后代；同时给予其他解一定机会避免早熟。
   - **相似性矩阵锦标赛选择（后期迭代）**：根据资产组合的重叠程度选择最相似的解配对，以微调权重分配。

3. **变异算子（三种类型）**：
   - **第一类**：对资产和权重进行微小扰动（正态分布），用于局部搜索。
   - **第二类**：根据收益、风险、收益风险比等指标从特定集合中替换资产，并重新随机分配权重。
   - **第三类**：生成全新投资组合，用于全局探索。
   - 不同阶段使用不同组合：前期多用第二、三类；后期多用第一类。

4. **Explorer算子**：针对修复后的个体（或随机个体），通过5种启发式权重分配方法（等权重、随机、基于波动率逆比、基于收益波动比、基于收益方差比）生成候选解，并利用公式（9）计算综合改进指标。然后在目标与候选之间的空间内随机探索最多Ω次，寻找支配性或非支配性改进解。主要用于调整修复后的投资组合权重。

5. **基数约束修复技术**：
   - **关联性技术（Associated-based）**：将当前Pareto前沿聚类，统计每个聚类中资产与违规个体中已有资产的共现频率，选择高频共现的资产加入。
   - **基于分数的技术（Score-based）**：预先通过K-means对资产聚类，计算聚类之间的低相关性得分（基于资产相关系数矩阵），选择得分最高的聚类中的资产加入。
   - **基于阈值的技术（Threshold-based）**：对违规个体中的每个资产，查找与其相关性低于阈值γ的资产候选，通过严格系数逐渐放宽标准，加入低相关资产以分散风险。

6. **边界约束处理**：首先通过算法10处理资产不足或权重越界问题（替换重复资产或添加新资产），然后通过算法11将超量的权重重新分配到其他可行位置，保证每个资产权重在[εi, δi]之间且总和为1。

### 算法流程概要（Algorithm 12）
1. 初始化随机种群（大小为|p|，编码长度为Kmax）。
2. 对每一代：
   - 如果为前期阶段（i < Π），使用膝点选择交配，使用第二、三类变异，生成后代A。
   - 如果为后期阶段，使用相似性选择，使用第一、二类变异，并激活Explorer算子（偶数代且修复个体不足时）。
   - 对违规个体进行基数约束修复（前期可能延迟激活关联性技术）。
   - 对所有个体执行边界约束修复。

3. 重复直到最大迭代次数。
