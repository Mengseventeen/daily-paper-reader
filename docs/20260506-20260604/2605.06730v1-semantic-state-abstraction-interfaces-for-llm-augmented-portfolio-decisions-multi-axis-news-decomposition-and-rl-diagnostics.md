## 摘要
我们提出语义状态抽象接口（SSAI）：一种方法论模板，用于将稀疏的非结构化文本映射为K个可审计的命名坐标，在无新闻日具有中性默认值，旨在分离序列决策系统中的表示假设与优化方差。我们的贡献在于框架及其评估协议，而非声称SSAI优于更密集的替代方案。我们在美国股票面板（30只纳斯达克100成分股，FNSPID新闻，2019-2023年测试）上以K=4个轴（情绪、风险、信心、波动率预测）实例化SSAI，并在共享相同固定φ的直接因子组合、监督岭预测器和RL智能体（DP-PPO、SAC）上进行评估。四因子组合实现了307.2%的累计收益率和1.067的夏普比率，但相对于买入持有（243.6%）的明显收益未能通过覆盖分层控制检验，在≥0.2%的成本下反转，且相对于仅情绪基线在统计上脆弱；在此设定下，PC1综合指标和FinBERT组合基线是更强的排序信号。Ridge和RL模块诊断了表示与优化器效应。我们将SSAI定位为一种可解释性-性能诊断工具和可复用的稀疏文本决策系统协议。

## Abstract
We introduce Semantic State Abstraction Interfaces (SSAI): a methodological template for mapping sparse unstructured text into $K$ auditable, named coordinates with neutral defaults on no-news days, designed to separate representation hypotheses from optimisation variance in sequential decision systems. Our contribution is the framework and its evaluation protocol, not a claim that SSAI outperforms denser alternatives.   We instantiate SSAI with $K=4$ axes (sentiment, risk, confidence, volatility forecast) on a US-equity panel (30 NASDAQ-100 names, FNSPID news, 2019--2023 test), and evaluate it across direct factor portfolios, supervised ridge forecasters, and RL agents (DP-PPO, SAC) that share the same fixed $φ$. The four-factor factor portfolio reaches 307.2% cumulative return and Sharpe 1.067, but apparent gains versus buy-and-hold (243.6%) fail coverage-stratified controls, reverse at $\geq 0.2$% costs, and are statistically fragile versus a sentiment-only baseline; a PC1 composite and a FinBERT portfolio baseline are stronger ranking signals in this setting. Ridge and RL blocks diagnose representation versus optimiser effects. We position SSAI as an interpretability-performance diagnostic and reusable protocol for sparse-text decision systems.

- **问题**：在LLM增强的序列决策系统（如强化学习交易）中，现有方法将文本编码为高维潜向量或单标量情感分数，导致无法区分性能提升是来自文本表示质量还是优化器噪声。单标量情感混淆了正交概念（如负面情绪与高风险本质不同）。

## 2. 方法论

- **核心思想**：定义SSAI（语义状态抽象接口），将无结构化文本映射为K个固定的、可审计的命名坐标，无新闻时以中性默认值填充。该接口冻结后共享给多种估计器，使得输出差异可归因于算法而非表示。
- **关键技术细节**：
    - **定义**：$\phi: N_{s,d} \to \mathbb{R}^K$，其中$N_{s,d}$是资产$s$在日期$d$的文章集合。$K=4$轴：sentiment（情感）、risk（风险）、confidence（信心）、volatility_forecast（波动预测），每轴整数范围1-5，中性默认值$[3,3,3,3]$。
    - **LLM评分**：通过固定提示和例文从冻结的GPT-4o提取四个整数分数，温度0，batch大小20。对40,850篇文章评分，成功率97.9%。
    - **聚合**：每个股票-日使用前3天窗口内文章的平均分。
    - **评估不变性**：由于$\phi$固定且共享，不同估计器的结果差异归因于算法，而非表示（附录A.1形式化）。
    - **状态向量**：$s_d = [c_d, p_d, h_d, f_d, \sigma_d] \in \mathbb{R}^{421}$，包括现金、价格、持仓、7个技术指标（30支股票）和4个LLM信号（30支股票）。
    - **RL奖励**：引入经回撤惩罚的回报：$r_t = \frac{\Delta W_t}{W_0}\lambda - \alpha \max(0, D_t)^2$，其中$D_t$是分式回撤，$\lambda=10^{-4}$，$\alpha=0.1$，惩罚回撤二次项。
    - **三种因子组合**：SFP（岭回归权重）、SRF（残差化非情感轴）、SCW（验证集softmax权重）。监督岭预测器、DP-PPO（PPO+回撤惩罚）、SAC（off-policy）。
- **算法流程**：新闻 → LLM评分（4轴）→ 股票-日聚合 → 填充到状态 → 输入共享$\phi$给因子组合/岭回归/RL → 分别评估，比较差异。

## 6. 主要结论与发现

- **因子组合**：4轴SFP累计收益307.2%（夏普1.067），超出B&H（243.6%）63pp，但此增益是**组合构成效应**而非信号质量：在覆盖分层（低/中/高）内，SFP均劣于同层B&H（表3），且在≥0.2%交易成本下反转（表12）。4轴 vs 仅情感日利差不显著（Wilcoxon p=0.556）。
- **PC1与FinBERT更强**：PC1-SFP（433.6% CR）和FinBERT-SFP（386.3%）均显著优于4轴SSAI，且PC1载荷近似等权（0.257/0.260/0.240/0.242），Softmax-SFP（等权标准化轴）也达439.8%，证明命名轴本身不损失性能，而ridge权重拖累。
- **监督岭回归**：朴素添加LLM列有害；高语义倾斜或VADER/TF-IDF可恢复价格基线约0.67的夏普；FinBERT-ridge仅0.649，与价格相当。
- **RL诊断**：SAC（夏普1.059）显著优于DP-PPO（0.920），但CR差异不显著（p=0.604），证明算法选择主导。DP-PPO中性掩码与全信号多种子均值几乎相同（Wilcoxon p≈0.25-0.31），信号贡献微弱且统计不足。
- **总结性结论**：SSAI作为可解释性-性能前沿的诊断工具，其可审计性代价（约126pp CR vs PC1）可被量化；性能主要来自组合构成和算法选择，而非语义表示本身。作者**不声称SSAI性能优越**，而是提供诊断协议。
